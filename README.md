# Screenshot_Organizer
Overview:
Screenshot Organizer is an AI-powered Android application that automatically organizes screenshots into categories such as Study, Travel, Social, Shopping, Food, and Others using OCR (Optical Character Recognition) and Generative AI.
The app helps users manage large numbers of screenshots efficiently and quickly access important screenshots using the Important ⭐ feature.

✨ Features:
📸 Automatically fetches screenshots from the device
🤖 AI-based screenshot categorization
🔍 OCR text extraction using ML Kit
📂 Categories: 
  1. Study
  2. Travel
  3. Social
  4. Shopping
  5. Food
  6. Others
⭐ Important/Favorite screenshots feature
🖼 Full-screen screenshot viewer
↔ Swipe navigation between screenshots
🗑 Delete screenshots directly from the app
🔄 Auto-refresh when new screenshots are added
🌙 Supports both Light Mode and Dark Mode

🛠 Technologies Used:
1. Kotlin
2. Jetpack Compose
3. Android Studio
4. ML Kit OCR
5. OpenAI API
6. MediaStore API
7. Coil Image Library

📱 How the App Works:
1. The app fetches screenshots from the device storage.
2. OCR extracts text from screenshots.
3. The extracted text is analyzed using Generative AI.
4. AI categorizes screenshots into appropriate folders.
5. Users can:
   View screenshots
   Swipe between screenshots
   Delete screenshots
   Mark screenshots as Important ⭐
